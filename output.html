第1章 开发环境配置

1.1 为什么要学C语言

C语言是计算机科学的基石：

  • 操作系统层：Linux/Windows内核用C编写
  • 系统编程层：驱动程序、嵌入式开发
  • 应用开发层：数据库(MySQL)、编译器(GCC)、游戏引擎
  • 学习基础层：理解计算机底层原理的最佳语言

相比于上学期所学的Python，C或许稍微复杂一点，但性能更好。

现代C语言标准持续演进：C11/C17/C23，依然是性能关键型应用首选。

目前广泛应用的大语言模型运行框架 llama.cpp 就是基于C/C++开发的，提供了比Python更强的性能。

如果更加关注内存安全性，可以进一步尝试golang或者rust。

千里之行，始于足下，积跬步以至千里。

1.2 主流C语言编译器对比

编译器开发商主要特点适用平台
**GCC**GNU项目历史最悠久，Linux默认，开源免费Linux/Unix为主，也支持Windows
**Clang**LLVM项目模块化设计，错误信息友好，编译速度快高性能、跨平台
**MSVC**MicrosoftWindows原生，Visual Studio集成Windows

本课程选用 GCC/Clang/LLVM，优势：

  • 更清晰的错误提示
  • 更好的诊断信息
  • 现代化工具链

1.3 LLVM项目架构

LLVM采用前后端分离设计：

前端（Front End）：

  • Clang (C)
  • Clang (C++)
  • Clang (Objective-C)

其实不仅有C。

中间表示（LLVM IR）：统一优化器

后端（Back End）：

  • x86
  • ARM
  • RISC-V

优势：统一优化器，跨平台代码优化。

1.4 开发环境安装

Windows系统安装

1. 打开 Windows 终端

2. 运行安装命令：

winget install llvm
尽量用包管理器，这样会在其他操作系统上也能得到完全一致的安装体验，也最为安全。
### Ubuntu系统安装
1. 打开 Bash 终端
2. 运行安装命令
sudo apt install llvm -y
注意，要用sudo来以root权限来运行。
### 验证安装
$ clang --version
clang version 17.0.0
Target: x86_64-w64-windows-gnu
或者
$ clang --version
Apple clang version 21.0.0 (clang-2100.0.123.102)
Target: arm64-apple-darwin25.4.0
Thread model: posix
InstalledDir: /Applications/Xcode.app/Contents/Developer/Toolchains/XcodeDefault.xctoolchain/usr/bin
思考一个问题，是不是版本号越高越好？

1.5 第一个C程序：Hello World

#include 
int main() {
    printf("Hello, World!\n");
    return 0;
}
**代码解析**：
- `#include ` - 包含标准输入输出库
- `int main()` - 程序入口函数
- `printf()` - 输出函数
- `return 0` - 返回0表示正常结束
始终记住：一切从主函数开始，也从主函数结束
##### 新手常见错误：中文标点，遗漏符号，拼写错误等等。
怎么办？将代码，以及运行时候的错误提示，以文本形式复制给AI开发助手，比如OpenLaoKe来询问和定位错误位置，并尝试修复。

1.6 编译过程四阶段

阶段命令输入输出主要工作
**预处理**`clang -E``.c``.i`宏展开、头文件包含
**编译**`clang -S``.i``.s`C代码转汇编
**汇编**`clang -c``.s``.o`汇编转机器码
**链接**`clang``.o``.exe`链接系统库

但是实际上，真正的开发场景中，大家往往就 一步完成：

$ clang hello.c -o hello.exe
大家不需要去背诵和记忆这些过程，而是想明白并且理解这个过程，
能把这个过程想明白，从结构上清晰理解了，就可以了。
我们要发展和培养的是理解流程并且复用于其他场景的能力，而不是某一个等级考试的题目解答之类的。创造力和想象力更重要，记忆力的考核不那么重要了。

1.7 常用编译选项

选项说明
`-o name`指定输出文件名
`-Wall`开启所有警告
`-Wextra`开启额外警告
`-g`生成调试信息
`-lm`使用 `math.h` 库
`-O2`优化级别2
`-std=c17`使用C17标准

推荐命令：

$ clang hello.c -o hello -Wall -Wextra -g -lm
这些细致的选项都不要去记，只要理解了，知道有这回事就可以了。
以后用的时候，随时让AI辅助编程助手来帮你翻译一下即可。
